# CERTH-HIT-Data-Federation
Built a semantic data federation prototype using DCAT-AP and SPARQL. Indexed mobility datasets from CERTH-HIT and queried them using GraphDB. Exploring semantic web and data engineering systems.
# Data Federation using DCAT-AP

This project demonstrates how mobility datasets can be indexed and discovered
through a shared metadata catalogue using DCAT-AP and SPARQL.

Technologies used:
- RDF / Turtle
- DCAT-AP metadata standard
- GraphDB triplestore
- SPARQL queries

Datasets:
- network-speed
- network-congestion
- itravel-paths

Goal:
Simulate data federation across mobility testbeds such as those in AIOTI.
